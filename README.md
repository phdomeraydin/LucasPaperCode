# Lucas–Catalan Based Enumeration and Performance Modeling

This repository provides Python source code used to generate
Tables 1–3 and Figure 2 in the accompanying scientific manuscript.

All numerical results are derived directly from closed-form
combinatorial formulas. No external data files 
are required, ensuring full reproducibility.
